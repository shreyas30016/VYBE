# VYBE AI

Your Personal AI Fashion Companion

## Problem

Choosing outfits every day is time-consuming and often leads to underutilized wardrobes.

## Solution

VYBE AI combines conversational AI, computer vision, weather intelligence, and wardrobe management to deliver personalized styling recommendations.

## Features

✨ AI Stylist
📸 Magic Scan
👕 Smart Wardrobe
🧳 Packing Planner
🌦 Weather-aware Outfits
📊 Wardrobe Insights
💬 Chat History
